serve: Command = ...

Serve the core My Elixir api

e.g.

my-cli serve --config-file agent.json

The configuration file is populated in a ConfigMap during Helm deployment. The default serverRoot is /api/v1. This strategy allows us to have multiple such api's running at any given time.
